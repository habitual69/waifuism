waifuism (Unofficial hanime.tv) API

Fetch content from hanime.tv to use with IPTV clients for seamless entertainment.

Endpoints

1. Trending Playlist

Generate a playlist based on trending content, with support for pagination.

GET /playlist/trending/:time/:page http://localhost:3000/playlist/trending/daily/1

Parameters:

  • time: The time frame for trending content (e.g., daily, weekly).
  • page: Page number for pagination.

2. Tag-Based Playlist

Create a playlist based on a specific tag, also supporting pagination.

GET /playlist/tags/:tag/:page http://localhost:3000/playlist/tags/3d/1

Parameters:

  • tag: The tag to filter content by.
  • page: Page number for pagination.

Using This API

To use this API, make a GET request to the desired endpoint with the appropriate parameters. The response will be an M3U8 playlist file that you can use directly in compatible IPTV clients for streaming content.
